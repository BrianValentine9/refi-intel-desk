# Evals

The daily morning brief (Step 5) is generated by an LLM from structured rate and
trigger-ladder data. Because hiring managers — and mortgage professionals — can't
trust numbers a model might hallucinate, every brief is checked against the source
data before it ships. This folder holds that eval harness: it verifies that figures
quoted in the brief (current 10Y Treasury, 30Y averages, the count of modeled loans
clearing net-tangible-benefit and recoupment at each trigger) exactly match the
computed inputs, and that the brief makes no unsupported directional claims. The goal
is a measurable accuracy bar, reported in the README, that proves the AI layer is
grounded rather than decorative.
